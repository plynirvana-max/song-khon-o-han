# 비자 정보 업데이트하는 법 (Jay & Toto 전용)

## 한 파일만 고치면 됩니다: `visa-data.js`

이 파일을 GitHub에서 열어 수정 → Commit 하면, 사이트 비자 페이지(양 언어)가 자동으로 갱신됩니다.
세 종류를 관리해요:

### 1) changes (최근 변경 타임라인)
새 변경이 생기면 `"changes": [` 바로 아래에 이 블록을 추가 (최신을 맨 위에):
```
{"date":"2026-08-01","tag":"E-9","vi":"(베트남어 설명)","ko":"(한국어 설명)","src":"https://공식출처링크"},
```

### 2) rumors (소문 팩트체크)
커뮤니티에서 도는 소문을 검증해 추가:
```
{"q_vi":"소문(베)","q_ko":"소문(한)","verdict":"false",
 "a_vi":"답변(베)","a_ko":"답변(한)","src":"https://출처"},
```
`verdict`는 셋 중 하나: `"true"`(사실) / `"false"`(거짓) / `"depends"`(경우에 따라)

### 3) official (공식 링크 허브)
바꿀 일 거의 없음. 필요시 name_vi/name_ko/url/note_vi/note_ko 수정.

## 마지막에 꼭: 맨 위 `"updated"` 날짜를 오늘로 바꾸기
```
"updated": "2026-08-01",
```

## 규칙
- 확실하지 않은 정보는 올리지 말 것. 공식 출처(hikorea.go.kr, 1345)로 확인 후에만.
- 모든 항목에 가능하면 `src`(출처 링크)를 넣기.
- 커밋하면 1~2분 내 사이트 반영.
